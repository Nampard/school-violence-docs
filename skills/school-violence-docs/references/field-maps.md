# 필드·셀 매핑 레퍼런스

> 이 매핑은 `assets/templates/`의 **OO고등학교 양식**에 맞춰져 있다. 표 인덱스·행 번호는 이
> 템플릿 기준으로 안정적이다. 다른 학교 양식을 쓰면 `parse_form`/구조 확인 후 인덱스를 조정해야 한다.
> 채움은 `scripts/`의 필러가 처리하므로, 보통은 JSON만 만들면 된다. 이 문서는 디버깅·확장용.

## 채움 방식 2종 (왜 나뉘나)
- **단순 라벨-값 행** → kordoc `fill_form`도 가능하지만, 필러는 일관성을 위해 lxml로 처리.
- **표 그리드·중첩표·체크박스** → `fill_form`이 골격을 덮어쓰거나 못 채운다. 반드시 lxml 셀주소 방식.

## 1. 사안접수 보고서 (`fill_jeopsu.py`)
표: `[0]`=상단 제목표, `[1]`=본문표, `[2]`=사실확인 중첩표.

| 항목 | 위치 | 비고 |
|---|---|---|
| 사안번호 | 제목셀 run `* 사안번호: OO고-2026-` 뒤 | `N호` 삽입 |
| 접수일시 | 본문표 cellAddr(3,1) | |
| 신고자(성명,신분) | (4,1) | |
| 접수·인지 경로 | (4,11) | |
| 관련학교명 | (18,3) | 공동사안 아니면 "해당없음" |
| 관련학생 그리드 | `가해관련`+`탈북학생` 포함 tr들, tc[0]학교·[1]학번·[2]성명·[3]성별·[5]비고 | 비고는 체크박스 |
| 사실확인 관련학생/일시/장소/내용 | 중첩표 행 0/1/2/3 의 tc[1] | |
| 사실확인 유형 | 중첩표 행4 tc[1] | `□항목`→`■항목` |

**유형 체크박스 매핑** (법적 효과 주의 — 사용자/조사자 분류 있을 때만):
| 입력 키워드 | 체크 |
|---|---|
| 폭행/신체폭력 | ■폭행 |
| 언어폭력·따돌림·강요·사이버폭력·금품갈취 | 각 항목 |
| 성희롱 | ■기타(성희롱) (전용칸 없음) |
| 성폭력 | ■성폭력 — 수사기관 통보의무 등 영향, 임의 체크 금지 |
| 기타 | ■기타 |

## 2. 전담기구 심의결과 보고서 (`fill_jeondamgigu.py`)
자유서술형. 표: `[0]`=본문, `[1]`=4요건표.

| 항목 | 방식 | 앵커/위치 |
|---|---|---|
| 사안번호 | 치환 | `2026- 호` |
| 심의주제 번호 | 치환 | `2026-0  호` → `2026-NN호` |
| 일시 | 치환 | `2026년   월   일 (  요일)` |
| 참석자 | 치환 | `3. 참 석 자 : ` 뒤 |
| 조사내용 | 치환 | 첫 `<hp:t>  • </hp:t>` |
| 결정사항 | 치환 | 첫 `<hp:t> • </hp:t>` |
| 4요건 1~4 해당여부 | lxml | 4요건표 rows[1].tc[2], rows[2/3/4].tc[1] |
| 동의서 제출여부 | lxml | 4요건표 rows[6].tc[1] |

> 치환으로 들어가는 자유서술 값은 `& < >`를 이스케이프해야 XML이 깨지지 않는다(필러가 처리).

## 3. 자체해결·종결 결과 보고서(통합) (`fill_result.py`)

## 분기 판정 (흐름 ②)
사안조사 보고서를 읽고 → 인터뷰로 확정:
- **자체해결**: 4요건 모두 O **그리고** 피해측 동의(O) → 전담기구 보고 + 자체해결 결과 보고
- **종결**: 학교폭력 아님(오인신고 등) → 전담기구 보고 + 종결 결과 보고
- **심의위 이송**: 4요건 중 하나라도 X **또는** 동의 미제출(X) → 전담기구 보고만 (결정사항에 이송 기재; 별도 이송 양식 없음)

> 4요건: (1)2주이상 진단서 미발급 (2)재산피해 없음/즉시복구 (3)지속적이지 않음 (4)보복행위 아님
> 공동학폭이면 전담기구는 연루 학교가 **각자 운영** → 자교 기준으로 작성.
