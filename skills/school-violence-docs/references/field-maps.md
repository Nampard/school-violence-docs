# 필드·셀 매핑 레퍼런스

> 이 매핑은 `assets/templates/`의 **OO고등학교 양식**에 맞춰져 있다. 표 인덱스·행 번호는 이
> 템플릿 기준으로 안정적이다. 다른 학교 양식을 쓰면 `parse_form`/구조 확인 후 인덱스를 조정해야 한다.
> 채움은 `scripts/`의 필러가 처리하므로, 보통은 JSON만 만들면 된다. 이 문서는 디버깅·확장용.

## 글자 겹침(overlap) 방지 — `reflow_paragraphs`
빈 셀 문단에는 `<hp:linesegarray>`에 vertpos=0짜리 `<hp:lineseg>`가 1개만 들어 있다(1줄 캐시).
긴 글자를 넣어도 이 캐시가 1줄 그대로면, 자동 줄바꿈된 2번째 줄부터 모두 같은 높이에 겹쳐 그려진다.
→ 모든 필러는 저장 직전 `H.reflow_paragraphs(body)`로 **텍스트가 있는 문단의 linesegarray를 제거**한다.
한글은 linesegarray가 없으면 열 때 칸 너비·폰트에 맞춰 줄 위치를 새로 계산하므로 겹침이 사라진다.
칸 너비(예: 사실확인 내용 `horzsize≈33124` vs 좁은 칸 `6300`)가 제각각이라, 고정 글자수 줄바꿈보다
이 방식이 견고하다. 새 필러를 만들면 저장 직전 반드시 이 함수를 거칠 것.

## 채움 방식 2종 (왜 나뉘나)
- **단순 라벨-값 행** → kordoc `fill_form`도 가능하지만, 필러는 일관성을 위해 lxml로 처리.
- **표 그리드·중첩표·체크박스** → `fill_form`이 골격을 덮어쓰거나 못 채운다. 반드시 lxml 셀주소 방식.

## 관련학생 칸 넘침 → 행 자동 복제
양식 기본 수용량은 **사안접수 4명 / 자체해결·종결 결과 4명**. 학생이 이를 넘으면 필러가 **마지막 학생행(블록)을
인원수만큼 표 안에서 복제**해 전원을 한 문서에 정식 서식으로 채운다(`hwpx_lib.clone_block_after`).
- 사안접수: 학생 1명=1행. 복제 후 `관련학생` 세로병합 라벨의 rowSpan도 자동 확장(`extend_rowspan_label`)해 빈 칸을 막는다.
- 자체해결·종결: 학생 1명=2행(헤더+데이터) 묶음. 복제 시 하위 고정행(사안조사내용·체크·관계회복)은 `off=2×추가인원`만큼 자동 이동.
- 복제 후 표 전체 `rowAddr`를 tr 순번으로 재번호(`renumber_rows`)하고 `rowCnt`를 갱신한다. → 별지·수기 추가 불필요.

## 4. 자체해결/종결 동의서 (`fill_donguiseo.py`) — 피해측 서명 문서
표 `[1]` 사용. `종류`로 자체해결/종결 구분. **모두 lxml로 채운다**(아래 fill_form 주의 참조).

| 항목 | 자체해결 | 종결 |
|---|---|---|
| 사안번호 | 셀(0,0) 안 `-2026-OO` 자리 → `-{연도}-{번호}` 정규식 치환(여러 run 분할 주의) | 셀(0,0) `-20○○-○○` 자리 |
| 관련학생(가해·피해 모두) | 행 2,4 의 col 1/2/3 = 소속학교/학년반/성명 | 행 2,4,6 |
| 사안내용 | 셀(5,1) | 셀(7,1) |
| 학교명/약칭 | `apply_school` (OO고등학교장 귀중 포함) | 동일 |

> **관련학생 기재**: 피해관련뿐 아니라 **가해관련 학생 인적사항도 모두 적는다**(서명은 피해측이 개인별로 함).
> **학년/반 형식**: 컬럼은 `학년/반`이므로 '1학년 3반'은 `"1/3"`으로 적는다(번호 미포함).

> **⚠️ fill_form 사용 금지(이 양식)**: kordoc `fill_form`은 "사안 조사 내용" 칸은 잡지만 ① 관련학생 그리드·사안번호는
> 못 채우고(신뢰도 0.33) ② **일부 한글을 손상**(예 "다툼이"→"다투이")시킨다. 피해 학생·보호자가 읽고 **서명하는
> 문서**라 글자 손상은 허용 불가 → 반드시 `fill_donguiseo.py`(lxml, 바이트 정확)로 채운다.
>
> **사안내용 문체**: 사안조사 보고서의 사실을 따르되 **완곡·객관적 표현으로 500자 내외**(보고서의 '최소 500자'와 다름).
> 가해 행위를 자극적·단정적으로 묘사하지 않는다(피해자 2차 가해 방지).

## 분기 판정 (흐름 ②)
사안조사 보고서를 읽고 → 인터뷰로 확정:
- **자체해결**: 4요건 모두 O **그리고** 피해측 동의(O) → 전담기구 보고 + 자체해결 결과 보고 + 자체해결 동의서(피해측)
- **종결**: 학교폭력 아님(오인신고 등) → 전담기구 보고 + 종결 결과 보고 + 종결 동의서(피해측)
- **심의위 이송**: 4요건 중 하나라도 X **또는** 동의 미제출(X) → 전담기구 보고만 (결정사항에 이송 기재; 별도 이송 양식 없음)

### 4요건·동의 칸 채우는 법 (★ 분기에 따라 다름)
4요건은 **충족 안 되는 경우가 많다.** 분기가 확정된 뒤 칸을 채운다:
- **자체해결 / 종결로 확정** → 4요건 4칸 + 동의 칸 **모두 `O`** (자체 처리되려면 요건·동의가 충족된 것이므로).
  data.json: `"요건": ["O","O","O","O"]`, `"동의": "O"`.
- **심의위 이송으로 확정** → 4요건·동의 칸을 **추측해서 채우지 않는다.** 인터뷰로 "어느 요건이 X인지 / 동의가 미제출인지"
  확인된 칸만 그 값으로 채우고, **답이 없으면 그 칸은 빈칸**으로 둔다(사용자가 한글에서 채움). 억지로 X를 찍지 않는다.

> 4요건: (1)2주이상 진단서 미발급 (2)재산피해 없음/즉시복구 (3)지속적이지 않음 (4)보복행위 아님
> 공동학폭이면 전담기구는 연루 학교가 **각자 운영** → 자교 기준으로 작성.
